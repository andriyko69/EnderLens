# Ender Lens [Picture Mode Addon]

**Ender Lens [Picture Mode Addon]** is a small addon for **Picture Mode** on **NeoForge 1.21.1**.

Instead of opening Picture Mode from the default in-game button, this addon adds an **Ender Lens** block.  
Right-clicking the block opens Picture Mode directly, giving you a more immersive and build-friendly way to access it.

## Requirements

- Minecraft **1.21.1**
- **NeoForge**
- **Picture Mode** by **Icanttellyou**
